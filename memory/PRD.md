# Top Brasil Grill - Website PRD

## Problem Statement
Website em português para o restaurante Top Brasil Grill, localizado em Camarate/Lisboa, especializado em comida amazônica brasileira.

## Architecture
- **Frontend**: React com Tailwind CSS, Framer Motion (animações), React Fast Marquee (carrossel)
- **Backend**: FastAPI (não utilizado - conteúdo estático)
- **Styling**: CSS customizado com paleta de cores da marca (verde floresta, dourado tropical, castanho terroso, creme)

## User Personas
1. **Brasileiros em Lisboa** - Procurando comida autêntica do Pará/Amazónia
2. **Portugueses curiosos** - Interessados em experimentar culinária brasileira
3. **Turistas** - Visitantes procurando experiências gastronómicas únicas

## Core Requirements (Static)
- [x] Hero section com imagem vibrante e CTAs
- [x] Secção Sobre com história do restaurante
- [x] Menu visual com destaques (grid Bento)
- [x] Cardápio digital completo com categorias
- [x] Carrossel de avaliações de clientes
- [x] Links externos para reservas (TheFork) e pedidos (Uber Eats)
- [x] Integração Google Maps
- [x] Footer com redes sociais

## What's Been Implemented (30/03/2024)
- Website completo em português
- Design responsivo mobile-first
- Navegação suave com animações
- Menu digital com 4 categorias (Entradas, Pratos, Sobremesas, Bebidas)
- Reviews carousel com Marquee
- Links Instagram e Facebook funcionais
- Google Maps com morada correta
